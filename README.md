# AI Crop Disease Detection System

## Problem Statement
Farmers often struggle to identify crop diseases at an early stage. Late detection leads to crop damage and reduced yield.

## Solution
This project uses Artificial Intelligence to analyze plant images and detect crop diseases. The system identifies the disease and provides possible causes and suggestions for prevention.

## Features
- Upload plant images
- AI-based disease detection
- Shows possible causes of the disease
- Provides prevention suggestions

## Technologies Used
- Python
- Machine Learning
- HTML / CSS
- Flask (Backend)

## How to Run the Project
1. Clone the repository
2. Install required libraries
3. Run the backend server
4. Open the web interface
